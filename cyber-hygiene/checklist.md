# Cyber Hygiene — чеклист

Привязка к [linux-brain-map](../linux-brain-map/00-index.md).

Отмечай `[x]` по мере выполнения. Каждый пункт = теория + одна команда.

## §1 Архитектура (модуль 01)

- [ ] Знаю версию ядра: `uname -r`
- [ ] Знаю сколько RAM/CPU: `free -h && nproc`
- [ ] Запускал `user_audit.sh` на своей машине

## §2 Загрузка (модуль 02)

- [ ] Проверил время загрузки: `systemd-analyze`
- [ ] Нет failed-сервисов: `systemctl --failed`
- [ ] Знаю, что стартует при boot: `systemctl list-unit-files --state=enabled`

## §3 Процессы (модуль 04)

- [ ] Нет незнакомых процессов в `ps aux --sort=-%cpu | head`
- [ ] Нет zombie: `ps aux | awk '$8~/Z/'`
- [ ] Знаю PPID подозрительных процессов: `pstree -p`

## §4 Память (модуль 06)

- [ ] RAM не забита постоянно: `free -h`
- [ ] Swap не активен при простое: `vmstat 1 3`
- [ ] Проверил OOM в логах: `dmesg | grep -i oom`

## §5 Файлы и права (модуль 09)

- [ ] `~/.ssh` = 700, ключи = 600
- [ ] `chmod 600 ~/.ssh/id_*`
- [ ] Запустил `hash_checker.sh --baseline /etc > ~/baseline.sha256`
- [ ] Нет лишних SUID: `find /usr -perm -4000 2>/dev/null`

## §6 Пароли и аккаунты

- [ ] Уникальный пароль на каждый сервис (менеджер паролей)
- [ ] 2FA включена: почта, банк, GitHub, Google
- [ ] Нет автологина root/sudo без пароля

## §7 Сеть (модуль 11)

- [ ] Сменён пароль роутера с заводского
- [ ] Wi-Fi: WPA3 или WPA2-AES, не WPS
- [ ] Прошёл `home-network-audit.md`
- [ ] Запустил `port_scanner.sh 127.0.0.1`

## §8 Резервные копии

- [ ] 3-2-1: 3 копии, 2 носителя, 1 офсайт
- [ ] Проверена восстановление (хотя бы 1 файл)
- [ ] Шифрование бэкапа (LUKS / VeraCrypt / cloud E2E)

## §9 Обновления

- [ ] ОС обновлена: `sudo apt update && sudo apt upgrade` (или аналог)
- [ ] Автообновления безопасности включены
- [ ] Запускал `cve_monitor.py openssh` (или свой пакет)

## §10 Фишинг и социнженерия

- [ ] Не открываю вложения от незнакомых
- [ ] Проверяю URL перед вводом пароля
- [ ] Знаю, как сообщить о фишинге
