# Аудит домашней сети

Модуль 11 (STN: Socket → TCP stack → Netfilter).

## 1. Роутер

| Проверка | Как | Статус |
|----------|-----|--------|
| Пароль admin ≠ factory default | зайти в 192.168.0.1 / 192.168.1.1 | [ ] |
| Firmware актуален | раздел обновлений | [ ] |
| WPS выключен | Wi-Fi settings | [ ] |
| UPnP выключен (если не нужен) | advanced | [ ] |
| Remote admin выключен | WAN settings | [ ] |
| Гостевая сеть отдельная | guest Wi-Fi | [ ] |

## 2. Wi-Fi

| Параметр | Рекомендация | Статус |
|----------|--------------|--------|
| Шифрование | WPA2-AES или WPA3 | [ ] |
| Пароль | 16+ символов, не фраза из словаря | [ ] |
| SSID | не раскрывает модель роутера | [ ] |
| 2.4 GHz | отдельный пароль или отключить если не нужен | [ ] |
